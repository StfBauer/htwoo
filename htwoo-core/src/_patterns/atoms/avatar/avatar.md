This avatar is show with a size of 64px and round borders. If you construct other avatars based on this please pass it a mugshot url and a size parameter.

* 24px - .hoo-avatar-24
* 32px - .hoo-avatar-32
* 40px - .hoo-avatar-40
* 40px - .hoo-avatar-40
