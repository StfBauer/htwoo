# HOOTime


## Arguments

Format value as hh:mm:ss. e.g. `(new Date()).toISOString().split('T')[1]`

<ArgTypes of={HOOTime} />

export const Template = (args) => <HOOTime {...args} />;

  </Story>
</Canvas>
