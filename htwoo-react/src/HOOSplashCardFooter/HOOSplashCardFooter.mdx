# HOOSplashCardFooter

This example shows using two `HOOButton` components as the child(children) of the `HOOSplashCardFooter` component.
In reality you can put whatever DOM elements you want inside it the component the component simply injects all the children into the DOM.

## Arguments

<ArgTypes of={HOOSplashCardFooter} />

export const Template = (args) => <HOOSplashCardFooter {...args}>
  <HOOButton type={HOOButtonType.Primary} label="Create Something" onClick={() => { alert("Clicked"); }} />
  <HOOButton type={HOOButtonType.Standard} label="Call to Action" onClick={() => { alert("Clicked"); }} />
</HOOSplashCardFooter>;
