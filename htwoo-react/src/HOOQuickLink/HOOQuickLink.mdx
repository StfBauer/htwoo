# HOOQuickLink

Predefined enum defined for the supported button types:

```ts
export enum HOOQuickLinkType {
  "Link",
  "Compact",
  "Grid",
  "Button",
  "Tile"
}
```

Style options for the Quick Link Button type component:

```ts
export enum HOOQuickLinkStyle{ 
  "Outline"="",
  "Filled"="filled",
  "None"="no-outline"
}

export enum HOOQuickLinkAlignment{
  "Left"="",
  "Center"="center"
}

export enum HOOQuickLinkButtonLines{
  "Normal"="",
  "One"="one-line",
}
```

Image Size options for teh Quick Link Tile type component:

```ts
export enum HOOQuickLinkImageSize{
  "Small"="",
  "Medium"="img-m",
  "Large"="img-l",
  "XLarge"="img-xl",
  "Fill"="img-fill",
}
```

When the Quick Link element is not going to be hosted in a SharePoint page and you are using this component in conjunction with the `HOOQuickLinkGrid`, you need to add the columnSpan element to tell the grid how many columns the element will span.
