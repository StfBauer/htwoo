# HOOAvatar

Predefined enum defined for the supported avatar sizes:
```ts
export enum HOOAvatarSize {
  Px16 = "16",
  Px24 = "24",
  Px32 = "32",
  Px40 = "40",
  Px48 = "48"
}
```
## Example using 32px size

export const Template = (args) => <HOOAvatar {...args} />;
