# HOOField

A HOOField component will isolate the child components passed to it looking for a HOOLabel, HOOInputDesc, and HOOValidationMsg component. All other components will be rendered in the order they're passed in between the HOOInputDesc and HOOValidationMsg components.

## Arguments

<ArgTypes of={HOOField} />

export const Template = (args) => <HOOField {...args}>
  <HOOLabel label='My Field' />
  <HOOInputDesc description='My field description' />
  <HOOText value="Temp"/>
  <HOOValidationMsg validationMsg='My Validation Message'/>
</HOOField>;
